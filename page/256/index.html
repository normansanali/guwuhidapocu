BIOS The ASRock Z270 Killer SLI features a graphical UEFI BIOS, much like every modern motherboard these days. The BIOS defaults to a simple, single page mode, called the “EZ Mode”. There is the “Instant Flash” option that allows the user to flash a BIOS available into a USB drive, but there is also an “Internet Flash” option that has the ASRock Z270 Killer SLI automatically connecting to the company’s servers, checking, downloading and flashing the latest stable BIOS instantly, without having to download it from inside an OS.
